# Subroutine Saving in DIRAC

## Overview

DIRAC now has two ways to save subroutines:

1. **Automatic**: LLM-generated subroutines are auto-dumped to `~/.dirac/lib/TIMESTAMP/`
2. **Manual**: Use `<save-subroutine>` tag or `:save` shell command

## Auto-Dump (LLM-Generated)

When you use `<llm execute="true">` and it creates subroutines, they are automatically saved to:

```
~/.dirac/lib/TIMESTAMP/SUBROUTINE_NAME.di
```

Example:
```xml
<llm execute="true">
  Create a greeting subroutine
</llm>
```

This automatically dumps to `~/.dirac/lib/2026-03-14T09-30-45/greet.di` with metadata:
```xml
<!--
  Created: 2026-03-14T09:30:45.210Z
  Generated by: LLM
  Prompt: Create a greeting subroutine
-->
<subroutine name="greet" param-name="string">
  <output>Hello, <variable name="name"/>!</output>
</subroutine>
```

## Manual Save (User-Created)

### Using `<save-subroutine>` Tag

**Default location** (timestamped):
```xml
<save-subroutine name="my-sub" />
<!-- Saves to: ~/.dirac/lib/2026-03-14T09-35-20/my-sub.di -->
```

**Named directory** under `~/.dirac/lib/`:
```xml
<save-subroutine name="my-sub" path="utils" />
<!-- Saves to: ~/.dirac/lib/utils/my-sub.di -->
```

**Explicit file path**:
```xml
<save-subroutine name="my-sub" file="./my-library/my-sub.di" />
<!-- Saves to: ./my-library/my-sub.di -->
```

**With format option**:
```xml
<save-subroutine name="my-sub" format="braket" />
<!-- Saves in bra-ket notation instead of XML -->
```

### Using `:save` Shell Command

In DIRAC shell:

**Default location**:
```
:save my-sub
# Saves to: ~/.dirac/lib/TIMESTAMP/my-sub.di
```

**Named directory**:
```
:save my-sub utils
# Saves to: ~/.dirac/lib/utils/my-sub.di
```

**Explicit file path**:
```
:save my-sub ./my-lib/my-sub.di
# Saves to: ./my-lib/my-sub.di
```

## Usage Patterns

### Quick Save After Creation

In shell:
```
|subroutine name=quick-test param-name=string>
  |output>Quick test: 
  |variable name=name />
  
:save quick-test
```

### Organized Library

```
:save greet utils
:save farewell utils
:save calculate math
:save format-date utils
```

Results in:
```
~/.dirac/lib/
  utils/
    greet.di
    farewell.di
    format-date.di
  math/
    calculate.di
```

### Project-Specific

```
:save my-helper ./project/dirac-lib/my-helper.di
```

## File Structure

Saved files include metadata comment:
```xml
<!-- Exported subroutine -->

<subroutine name="greet"
            description="Greet a person"
            param-name="string:required:Person name">
  <output>Hello, <variable name="name"/>!</output>
</subroutine>
```

## Next Steps

The `~/.dirac/lib/` directory serves as a **staging area**. You can:

1. **Review** periodically: `ls ~/.dirac/lib/`
2. **Promote** useful ones to permanent library
3. **Index** for search: `:index ~/.dirac/lib/utils`
4. **Load** when needed: `<load-context query="greeting" />`
